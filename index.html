DOCTYPE is the document declaration, an instruction to the browser to parse this document as an HTML document.All HTML documents must start with a declaration. The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

The HTML element represents the content of an HTML document. There can be only one body element in a document.

The HTML class attribute is used to specify a class for an HTML element.

The class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.

The HTML id attribute is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.
